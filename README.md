# gopalbicycles-prog.github.io
Gopal Bicycles Website

## Local development

Serve the site over HTTP so paths like `/auth/login.html` work:

- **Python:** `python3 -m http.server 8000` → open http://localhost:8000
- **Node:** `npx serve . -l 3000` → open http://localhost:3000

Stop the server with Ctrl+C.
